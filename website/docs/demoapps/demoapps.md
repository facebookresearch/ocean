# Demo Apps

Ocean ships with a catalog of demo applications that exercise the framework's features and serve as starting points for your own projects. The demos are grouped into two collections:

- [**Cross-Platform Apps**](crossplatformapps/console.md) — applications that run on desktop and mobile platforms (macOS, Linux, Windows, iOS, Android). They cover the most common Ocean use cases: camera input, image and video preview, feature tracking, image alignment, line and point detection, and AR scene rendering with Shark.

- [**Quest Apps**](questapps/questapps.md) — VR demos targeting Meta's [Quest](https://www.meta.com/quest/) headsets, built on top of the [OpenXR](https://www.khronos.org/openxr/) framework. The collection starts with template applications that introduce OpenXR fundamentals and progresses through fully functional demos covering rendering, hand interaction, audio capture, panorama viewing, and more.

Each demo page lists the platforms it supports and provides step-by-step build instructions. If you are new to Ocean, the [Console](crossplatformapps/console.md) demo is the simplest place to start — it has no graphical surface and is the cleanest illustration of how to bootstrap an Ocean application.

For instructions on building the Ocean libraries and third-party dependencies that the demos depend on, see the [Building](../building/building.md) section.
